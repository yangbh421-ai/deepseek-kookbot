# KOOK 角色分配机器人

通过表情反应自助分配角色的 KOOK 机器人。

## 功能
- 管理员发送 `!sendcard` 指令，机器人在指定频道发送一张角色分配卡片。
- 用户在该卡片消息上添加 🖱️ 或 🎮 表情，自动获得/移除对应角色。
- 每 24 小时自动同步，确保反应与角色完全一致。
- 只处理机器人发送的特定卡片消息上的反应，不影响其他消息。

## 部署步骤

### 1. 创建机器人并邀请到服务器
- 在 [KOOK 开发者平台](https://developer.kookapp.cn/) 创建机器人，获取 Token。
- 将机器人邀请到你的服务器，并授予 **管理角色** 权限（必须）。

### 2. 获取必要 ID
- **服务器 ID**：开启开发者模式，右键服务器 → 复制 ID。
- **角色 ID**：右键角色 → 复制 ID（需要提前创建好“PC平台”和“主机平台”两个角色）。
- **频道 ID**：右键频道 → 复制 ID（用于放置卡片消息）。
